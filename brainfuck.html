Brainfuck is a language that only uses <[+-.,]>. Other characters are ignored
The languages works by initializing a 30,000 Element long array of ascii characters. A pointer stores the current index in that array
<Esc>: stop program
'<': pointer--
'>': pointer++
'+': increment current value
'-': decrement current value
'.': print current value as ascii character
',': read value into current value
'[', ']': loop -> repeats until current value is 0
