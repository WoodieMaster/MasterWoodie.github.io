"ShortO" is a Stack-based esoteric Programming Language created to make outputting text very easy
- Stack: any value used from the stack will automatically be removed from the stack - To print text to the console, just type the text
- To have some more functionalities you can create a code block by using $
- A code block is terminated by a ; and can be used to do the following:
 + To push a number onto the stack just write the number
 + To push a string onto the stack write the text surrounded by "
 + +: add the second from top value to the to value off the stack (Strings concatenate)
 + - remove the top value from the second from top value on the stack (number reduce length of string, string removes its value from other string)
 + *: multiply the top value with the second from top value off the stack (strings repeat number times, string and string produce the number 0)
 + /: divide the second from top value by the top value off the stack (string produces the number 0)
 + %: get the remainder of dividing the second from top value by the top value on the stack (string produces 0)
 + ,: print the top value of the stack
 + X: remove the top value of the stack
 + .: read a character from the user and put it onto the stack
 + !: negate the top value from stack
 + ': turn a number into the according unicode character
 + ~: turn a string number into an actual number
 + ^: repeat the top value of the stack (top value -> amount, second value -> repeated value)
 + S: turn a number into its string form
 + N: split each character from a string into its unicode value
 + >: skip following tokens if checked value is true (top value -> amount, second value -> check)
 + =: compare the second from top value with the top value (0: values equal, -1: top value greater, 1: bottom value greater; strings are always bigger than numbers)
 + v: reverse n amount of values (n: top value)
 + #: jump to a marker with the same name (name: string after #, terminated by ;)
 + D: remove n amount of characters from the output (n: top value (string as its length))
 + _: reads a string and adds it to the stack
 + T: checks the type of the top value from the stack. 0:number, 1:string
 + C: clear console
 + L: determines the length of the top value from the stack (numbers: as positive int)
 + |: checks if at least one of the top two values is true
 + &: checks if both of the top two values are true
- markers are defined positions inside of the code where the program can jump to.
- #: define a marker with a name (name: string after #, terminated by ;)
- \: print the following character to the console regardless of the type ($, #, \)
- Tokens are blocks of code that have a defined functionality
- Every token has a type and a value. following types exist:
- Booleans don't exist, so 0 and empty string are considered false, any other value is considered true
 + Text: one continues line of text without special characters create a single token
 + Marker do NOT create a token, however they will split the Text tokens
 + Value: strings ore numbers defined in a codeblock, they will automatically be added to the stack
 + Operator: any single-character instruction possible in the codeblock
 + Jump: a marker defined in a codeblock creates a jump token
